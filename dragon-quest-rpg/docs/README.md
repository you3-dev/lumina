## プロジェクト概要

### ゲーム情報

| 項目 | 内容 |
|------|------|
| タイトル | ドラゴンクエスト 〜勇者の冒険〜 |
| ジャンル | 2D RPG（ドラクエ風） |
| プラットフォーム | Webブラウザ（PC・モバイル対応） |
| 技術 | HTML5 + CSS3 + Vanilla JavaScript |
| コード規模 | 約12,000行（単一HTMLファイル） |

### 主な特徴

- 📱 **モバイル対応**: タッチ操作対応のバーチャルDパッド
- 💾 **オートセーブ**: LocalStorageによる進捗保存
- 🔊 **サウンド**: Web Audio APIによるBGM・SE
- 🗺️ **外部マップ**: JSON形式のマップデータ（55ファイル）
- ⚔️ **パーティシステム**: 最大4人のパーティ編成

## クイックスタート

### 起動方法

```bash
# ローカルサーバーで起動（例: Python）
cd dragon-quest-rpg
python -m http.server 8000

# ブラウザでアクセス
# http://localhost:8000/
```

### 操作方法

| 操作 | キーボード | タッチ |
|------|------------|--------|
| 移動 | 矢印キー | Dパッド |
| 決定 | Enter / Z | Aボタン |
| キャンセル | Escape / X | Bボタン |
| メニュー | Escape / X | Bボタン |
| マップ表示 | M | 📍ボタン |
